# RGIT Attendance Tracker

A web-based attendance tracker for RGIT Computer Engineering (Div F) students. This application allows students to log in with their roll numbers, track their attendance for various subjects, and view their overall progress.

## Features

* **Login with Roll Number:** Securely access your attendance data by entering your RGIT roll number.
* **Remember Me:** Option to keep your login session active for convenience.
* **Dashboard:**
    * View your overall attendance percentage.
    * See a breakdown of attendance by subject.
    * Track progress on a timeline.
* **Subjects Management:**
    * Add, edit, and delete subjects.
    * View attendance details for each subject.
* **Data Reset:** Option to clear all your tracked data.
* **Share Stats:** Share your attendance statistics easily.
* **Responsive Design:** Works seamlessly on various devices, from desktops to mobile phones.

## Technologies Used

* **HTML5:** For the structure of the web pages.
* **CSS3:** For styling and layout, including custom themes and responsive design.
* **JavaScript:** For interactive features, data management, and application logic.
* **Bootstrap Icons:** For a variety of icons used throughout the application.
* **Google Fonts (Inter):** For a clean and modern typography.

## How to Use

1.  **Clone the repository:**
    ```bash
    git clone <your-repository-url>
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd rgit-attendance-tracker
    ```
3.  **Open `index.html` in your browser** to access the application.

*Note: If you are deploying this to GitHub Pages, the application will be accessible at `https://vaibhavs-17.github.io/rgit-attendance-tracker/`.*
